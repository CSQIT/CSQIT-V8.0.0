## 📖 概述

CSQIT (Complete Spacetime Quantum Information Theory) 是一个从第一性原理出发的万物统一理论，基于六条公理，以"单子-连续谱-相互作用"为唯一本体论起点，系统性地推导出从微观粒子到宏观宇宙的所有可观测结构。

### 核心公理

| 公理 | 描述 |
|------|------|
| **公理1（存在性）** | 存在一个至多可数的单子集合 ℳ，以及一组基本相互作用 ℐ，构成连通关系网络 |
| **公理2（自生成性）** | 相互作用构成一个有限生成的 **operad** 结构 |
| **公理3（因果性）** | 存在局部有限的偏序 ≺，定义因果关系 |
| **公理4（量子概率性）** | 每个相互作用对应复数振幅 A_Φ = e^{iS}，满足 |A_Φ| ≤ 1 |
| **公理5（测度刚性）** | 同构的相互作用网络具有相同的总振幅 |
| **公理6（内部连续谱）** | 每个单子存在自然测度 μ_M，满足归一化和不变性 |
